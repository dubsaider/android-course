# Lesson 01 Demo (Android, Jetpack Compose)

В этой директории находится демо из первого урока: простое приложение на Jetpack Compose с двумя экранами и базовой навигацией.

## Что показывает демо

- запуск UI через `MainActivity` и `setContent`;
- базовую навигацию между экранами (`Home` <-> `Profile`);
- работу со `state` в Compose (счетчик кликов);
- простые элементы интерфейса: `Text`, `Button`, `Column`, `Row`.

## Структура

- `demo/MainActivity.kt` - точка входа приложения;
- `demo/navigation/AppNavigation.kt` - маршруты и навигационный граф;
- `demo/screens/HomeScreen.kt` - главный экран со счетчиком и переходом в профиль;
- `demo/screens/ProfileScreen.kt` - экран профиля с кнопкой возврата.

## Как запустить

1. Создайте Android-проект в Android Studio с поддержкой Jetpack Compose.
2. Убедитесь, что пакет проекта совпадает с `com.example.lesson01` или скорректируйте package/import в файлах.
3. Скопируйте файлы из папки `demo` в соответствующие директории модуля `app/src/main/java/...`.
4. Запустите проект на эмуляторе или устройстве.

## Как взаимодействовать с демо

1. На экране `Home` нажимайте `+1`, чтобы увеличивать счетчик.
2. Нажмите `Сброс`, чтобы обнулить значение счетчика.
3. Нажмите `Перейти в профиль`, чтобы открыть второй экран.
4. На экране `Профиль` нажмите `Назад`, чтобы вернуться на главный экран.
